D'Input is a package that provide input widget to build faster input with default border style

## Features

- Input concise
- Input Label
- Input Title
- Input Validator
- Input Area
- Some Style

## Usage

Some Simple Input
```dart
DInput(controller: TextEditingController()),

DInput(
    controller: TextEditingController(),
    hint: 'Hint 3',
    label: 'Label 3',
),

DInput(
    controller: TextEditingController(),
    hint: 'Hint 5',
    isRequired: true,
    title: 'Title 5',
),

DInput(            
    controller: TextEditingController(),
    maxLine: 4,
    minLine: 1,
    hint: 'Area',
),
```
